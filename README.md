# FlipItNews NLP Classification using Machine Learning and DistilBERT

## Project Overview

This project develops an automated news classification system capable of categorizing news articles into predefined categories using Natural Language Processing (NLP), Machine Learning, and Transformer-based models.

The project compares traditional machine learning techniques with modern Transformer architectures to evaluate their effectiveness in text classification.

---

## Business Problem

News platforms receive large volumes of articles daily. Manually categorizing articles is time-consuming and prone to inconsistencies.

The objective of this project is to build an automated classification system that can accurately assign articles to the correct news category, improving content organization and user experience.

---

## Dataset

**Dataset:** FlipItNews Dataset

The dataset contains news articles belonging to multiple categories, including:

* Business
* Sports
* Technology
* Politics
* Entertainment
